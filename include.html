Overview:

  • Open source web application for automated video creation.
  • Uses ari, an R package, for generating videos from text and images.
  • Provides an intuitive and user-friendly interface.

How Loqui Works:

  • Inputs: Google Slides URL or Microsoft PowerPoint file.
  • Extracts speaker notes from slides and converts them into audio.
  • Converts Google Slides to images.
  • Generates an mp4 video file with images and corresponding audio.
